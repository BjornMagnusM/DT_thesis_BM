# Duckietown Thesis 



[Duckietown](http://duckietown.com/) self-driving car simulator environments for OpenAI Gym.


## General dsl command 


This version has been refactored to support:

    Gymnasium API: Updated for compatibility with modern RL libraries.

    Hardware Acceleration: Optimized for NVIDIA GPUs (GTX 1650/RTX series).


See available duckiebots 
    dts fleet discover

Shutdown duckie bot 
    dts duckiebot shutdown  duckiebot14

Open duckietown dashboard 
    dts duckiebot dashboard  duckiebot14

Standard password for duckiebots 
    quackquack
